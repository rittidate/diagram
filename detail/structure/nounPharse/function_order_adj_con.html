Function and Type Modifiers

PRE-POSITION MODIFIER POST-POSITION MODIFIER

Modifiers for function or type may be placed before the noun. Function modifiers, mostly occur as -ing verb forms. Type modifiers often occur as past participle modifiers.

Function and type may also be expressed with a prepositional phrase: function by type with + noun.

FUNCTION NOUN FOR VERB-ING

Driving classes begin next week.

Classes for driving begin next week.

We'll be driving an off-road vehicle. vehicle (N) – auto, car

We'll be driving a vehicle for off-road driving. (SUV)
TYPE NOUN WITH + NOUN

Sugar cookies are my favorite.

Cookies with sugar are my favorite. (made with)

Whitney bought some laced shoes. (shoes with ties)
Whitney bought some high-heeled boots.

Whitney bought some shoes with laces.
